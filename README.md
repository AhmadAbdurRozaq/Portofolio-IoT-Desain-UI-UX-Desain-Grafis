## Portofolio Ahmad Abdur Rozaq ##

Repositori ini berisi kumpulan portofolio saya di keahlian tambahan, mulai dari **Internet of Things (IoT)**, **Desain UI/UX**, hingga **Desain Grafis**.  
Portofolio ini dibuat untuk menunjukkan hasil karya dan pengalaman saya dalam pengembangan teknologi sekaligus kreativitas di bidang visual.

Memulai usaha mandiri di bidang desain grafis sejak tahun 2021 pada masa pandemi, sebagai cara untuk mengisi waktu luang sekaligus mengasah keterampilan kreatif selama menjadi mahasiswa. Melayani pembuatan desain ilustrasi vector, logo, produk, dan berbagai kebutuhan visual lainnya sesuai permintaan pelanggan. Meski sempat berhenti sejenak karena fokus menyelesaikan studi, saya melanjutkannya kembali sebagai sarana produktif sambil mencari peluang kerja sesuai bidang saya sebagai lulusan Teknik Informatika.
